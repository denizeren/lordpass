How it works?

Remember only one secret key and let javascript do everything else on the client side.



Definitions

Secret word is your secret key to be used for all sites.(And you only have to remember this one!)

Site is the site you want to log in using your shiny generated password.

Version is added for precaution. If some day site leaks passwords or for some reason you want to change your password you only have to increment your passord version. And you will have completely different password.

Username is optionally added. If for some reason you have different usernames for a site, just add username value and different passwords will be generated for your different usernames.

Length helps you to crop N sized password starting from the beginning.

Alphabet is the coolest feature. It allows you to create a string from generated hash. And this string has only the characters which exist in the alphabet. So you can add as much different characters as you want.


Since everything is done on the clientside our server does not keep any of your data. And your password is securely generated.
